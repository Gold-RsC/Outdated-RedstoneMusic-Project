# Outdated-RedstoneMusic-Project
个人的红石音乐项目，最晚更新时间为2021年5月3日(追梦赤子心)

## 1.为何公开
丰富一下自己的主页(～￣▽￣)～

## 2.关于许可证
选择`Apache-2.0`不是为了保护专利，而是为了避免使用`MIT`而使大量无知者盗用乃至滥用代码，让圈子内产生许多粗制滥造的红石音乐（虽然圈子烂完了）

## 3.为何 Outdated 
- 时间上过时：如`README.md`第一句话所说，本项目最晚更新时间为2021年5月3日。
- 游戏版本上过时：本仓库中所有项目版本均为基岩版，且版本落后，指令、函数包等若干必要文件过时。
- 开发软件上过时：本仓库中所有项目均使用DevC++编写，为应对DevC++的若干bug，本项目使用若干下三滥的手段（包括但不限于无数个public/private、模板函数的奇怪写法，长达1.5分钟的编译时间）。
- 语言上过时：部分项目使用C++11编写，但是大部分项目仍旧是旧写法，优化极少。
- 能力上过时：学习程序生成红石音乐时，编程年龄仅仅4个月，生理心理年龄都极小，有能力才怪了
